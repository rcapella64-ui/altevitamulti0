ALTEVITA MULTI — UM PROGRAMA RESERVADO

Multi

Para quem você ama,
o cuidado mais completo que existe.

i A verdade que ninguém diz

Toda família que confia o cuidado de um pai ou de uma mãe a outras mãos se faz, em silêncio, a mesma pergunta: estou fazendo o suficiente?

O Altevita Multi nasceu para que essa pergunta, enfim, encontre repouso.

resposta

Sim. Você está.

ii O que muda quando o cuidado é completo

Há cuidado.

E há cuidado completo.

A diferença está no que se vê e, sobretudo, no que se sente.

Para poucos. Por escolha.

O Multi não é para todos.
É para quem deseja
o cuidado mais inteiro que existe.

Um programa reservado a famílias que entendem que presença, escuta e delicadeza não são detalhes são tudo.

iv — os cinco olhares

Os cinco olhares
sobre quem você ama.

01

Psicologia

Para que cada história continue sendo contada.

02

Nutrição

Para que cada refeição seja celebração, não prescrição.

03

Fisioterapia

Para que o corpo continue sendo casa.

04

Terapia Ocupacional

Para que os dias tenham cor, forma e propósito.

05

Fonoaudiologia

Para que cada palavra e cada refeição sigam seguras.

v Um time inteiro. Uma única pessoa.

Toda semana, os cinco olhares do Multi se encontram em torno de uma única intenção:

Cuidar de quem você ama.

Não há dois planos iguais porque não há duas histórias iguais.

vi — a jornada

A jornada.

  1. I

    A Escuta

    Os primeiros dias são para conhecer. Para escutar quem você ama, em silêncio e sem pressa. Cada olhar do time se aproxima, com delicadeza.

  2. II

    O Plano Único

    Um plano nasce, escrito a muitas mãos. Pensado para uma única pessoa. Entregue, em mãos, a você.

  3. III

    O Cuidado Contínuo

    Os dias seguem, acompanhados de perto. E, todo mês, uma carta chega à sua casa — para contar como ela ou ele tem florescido.

  4. IV

    O Reencontro

    A cada três meses, o time inteiro se reúne com você. Para celebrar caminhos. Para ajustar gestos. Para seguir, juntos.

vii — o que você recebe

O que você recebe.

I.

O Dossiê

Uma pasta entregue em suas mãos. Dentro, o plano único pensado para quem você ama, com a palavra de cada olhar do time. Para guardar. Para reler. Para mostrar a quem também ama.

II.

A Carta Mensal

Toda última sexta-feira do mês, uma carta chega. Com as pequenas conquistas. Com as palavras de quem esteve ao lado. Um gesto que vira ritual.

III.

O Encontro Trimestral

A cada três meses, o time se reúne com você em casa ou por vídeo. Não para falar de números. Para conversar sobre alguém querido.

Cuidado que você sente. E também vê.

viii — o que uma família leva

Você não está contratando um cuidado.

Você está escolhendo dormir mais tranquilo à noite.
Escolhendo ter o que contar aos seus irmãos.
Escolhendo saber, no fundo do peito, que está fazendo o melhor possível por quem te criou.

O Multi entrega, no fundo,
uma única coisa: paz.

ix — a diferença que não se explica

Famílias que escolhem o Multi
raramente voltam atrás.

Não por um contrato. Por um cuidado que, uma vez sentido, não se troca.
